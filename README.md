# VoiceRecorder
A simple voice recorder app created to try out voice recording and audio playing on SwiftUI.

**Requirements**
* Xcode 13 or higher
* iOS 15 or higher

**Technologies**
* SwiftUI
* AVFoundation
* CoreData
